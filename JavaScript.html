Understanding JavaScript with Māori Analogie

If you've ever tried to understand JavaScript but felt like it was written in an alien language, you're not alone. Learning to code can feel overwhelming, but think of it like learning how to cook. At first, it’s confusing, but once you understand the basics, you can start experimenting and making your own recipes!.

1. JavaScript, HTML, and CSS – The Hāngī

Think of a hāngī.

  • HTML: HTML is like the hāngī pit and stones. It provides the structure and foundation.
  • CSS:is like the seasoning, arrangement of food, and the way the meal looks. It makes things visually appealing.
  • JavaScript:is the fire that heats the stones and cooks the food. It brings everything to life and makes it work.

Without fire (JavaScript), the food wouldn’t cook, just like how a webpage without JavaScript would be static and lifeless.

2. Control Flow & Loops – Preparing a Hāngī

Control flow follows the steps of making a hāngī:

      let steps = ["Dig hole", "Heat stones", "Wrap food", "Place food in pit", "Cover and cook"];
      for (let step of steps) {
          console.log("Step: " + step);
      }

Loops automate repeated tasks, like wrapping multiple food items.

4. Arrays vs Objects – Waka vs Whānau

Arrays (waka): Ordered list of people.

      let waka = ["Hone", "Mere", "Tama"];
      console.log(waka[0]); // Hone (first person in the waka)

Objects (whānau): Named members.

      let whanau = { father: "Hone", mother: "Mere", son: "Tama" };
      console.log(whanau.mother); // Mere

5. Functions – Karakia Before a Hāngī

A function is like a karakia, preparing everything before starting.

      function karakia() {
          console.log("E te atua, whakapaingia ēnei kai.");
      }
      karakia();
